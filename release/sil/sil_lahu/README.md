SIL Lahu Keyboard
=====================

Copyright (C) 2010-2018 Here's Life Australia, SIL International

__DESCRIPTION__

This package will allow you to type Lahu using Charis SIL font.

There are six tone marks used in Lahu. These are on the `?` `/` `;` `:` `[` and `{` keys.
