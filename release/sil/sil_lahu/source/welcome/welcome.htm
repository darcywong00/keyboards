This keyboard was developed to facilitate typing text written in Lahu.

Lahu is spoken by over half a million people in China, Laos, Myanmar, Thailand and Vietnam. There are several orthographies in use. This one uses:

/ ˉ / high rising U+02C9 MODIFIER LETTER MACRON
/ ˇ /  high falling U+02C7 CARON
/ / mid (not marked)    
/ ˬ / low falling U+02EC MODIFIER LETTER VOICING
/ ˍ /  very low U+02CD MODIFIER LETTER LOW MACRON
/ ï¿½ / high checked U+02C6 MODIFIER LETTER CIRCUMFLEX ACCENT
/ ꞈ / low checked U+A788 MODIFIER LETTER LOW CIRCUMFLEX ACCENT

How to type in Lahu

This keyboard layout was developed in conjunction with several Lahu typists. The keyboard layout is a Unicode based keyboard layout for Tavultesoft Keyman.
